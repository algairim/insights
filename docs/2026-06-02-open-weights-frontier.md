# The Open Models Caught Up

`2026 June 2`

For a long time the best AI lived behind an API and a meter that ran on every call. That assumption broke quietly in 2026. [Google released Gemma 4 under Apache 2.0 at frontier benchmarks](disclaimer.md). [China's GLM-5.1 topped a coding leaderboard trained without a single Nvidia chip](disclaimer.md). [Alibaba's Qwen3](disclaimer.md) and [DeepSeek's V4](disclaimer.md) pushed open weights into territory that used to require a subscription.

What this means in practice is that [a model whose weights are public](disclaimer.md) can now do work that, a year ago, only a paid frontier API could. For a business handling sensitive data or running at volume, the maths changes: capable AI that runs on your own hardware, with no per-token bill and no data leaving the building. [The wider model landscape](2026-05-31-model-landscape-2026.md) now has a serious open tier underneath the famous names. The expensive option is no longer the only good one.
