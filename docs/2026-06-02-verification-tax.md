# Generation Got Cheap. Checking Didn't.

`2026 June 2`

There is a cost in every AI deployment that no one puts on the invoice.

Generating the output is nearly free now. Reading it carefully enough to know whether it is right is not. [That gap has a name](disclaimer.md) — the work moves from producing to checking, and checking does not scale the way generation does. A model can draft a hundred contracts in a minute. A human still has to read them.

This is not abstract. In one 2026 benchmark, [59% of agentic AI in production produced fabricated claims](2026-05-31-agentic-ai-failures.md), and [fluent, confident, sometimes-wrong output is the default failure mode](2026-05-31-ai-hallucination-risk.md), not the exception. The businesses getting value from AI are the ones that budgeted for the verification, not just the generation. [Testing whether an agentic system actually does what it claims](disclaimer.md) is now a line item, not an afterthought. The cheapest part of AI is making the output. The expensive part is trusting it.
