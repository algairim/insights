`2026 June 2`

Something shifted in 2026 that is easy to miss if you are only watching for new models.

[Microsoft turned Windows into an agent platform](disclaimer.md) and shipped [Agent 365](disclaimer.md) to manage agents the way IT departments manage employees. [Google shipped a complete four-layer agentic stack](disclaimer.md). [Salesforce put an agent inside every search bar](disclaimer.md). The agent is no longer an app you open. It is becoming part of the floor you stand on.

The [move from chatbots to systems that act](2026-05-31-agentic-shift.md) is now arriving as platform infrastructure, not as a product you choose. That carries a consequence most owners have not priced in: when agents are built into the operating system, the CRM, and the office suite, the question is no longer "should we adopt one." It is "do we know what the ones we already have are allowed to do." [The SDKs that turn a model into something that plans and acts](disclaimer.md) are now the default layer, not the frontier.
